8. 我希望把所有结果的元数据全部下载下来。单次下载上限是500条数据，因此检索结果需要多次下载，每次下载500条数据，直到所有数据都被下载下来。下载方式具体如下
（1）点击`删除`svg按钮，将历史下载勾选记录删除。弹出框点击`确认`
```
<span class="selected-count"><a href="javascript:;" class="slected">已选<span data-topcount="topcount">0</span>条</a><a href="javascript:;" title="清空已选文章"><i class="icon-del"></i></a></span>
```

```
<div class="layui-layer-btn layui-layer-btn-"><a class="layui-layer-btn0">确定</a><a class="layui-layer-btn1">取消</a></div>
```
（2）复选框`全选`勾选，翻页直到选择了500条上限数据，具体元素如下：
```
<span class="select-all" title="全选">
            <input type="checkbox" name="selectArticleAll" lay-filter="selectArticleAll" lay-skin="primary" title=""><div class="layui-unselect layui-form-checkbox" lay-skin="primary"><i class="layui-icon layui-icon-ok"></i></div>
        </span>
```

```
<a href="javascript:;" class="slected">已选<span data-topcount="topcount">20</span>条</a>
```

翻页按钮见`7 检索结果数量与页数`中的分页按钮

（3）点击`导出题录`
```
<a class="layui-btn layui-btn-xs behavior-exporttitle" data-key="export" href="javascript:void(0)" style="">导出题录</a>
```
（4）新页面中，`Excel导出`
```
<li data-type="excel" class="">Excel导出</li>
```
导出下载为excel文件
（5）同一个页面点击`参考文献`，txt文本文件
```
<li data-type="abstract" class="layui-this">参考文献</li>
```
（6）此批次导出的参考文献样例如下
```
[1]汪沛,马梅,罗文慧.从询唤到“制造”:青年的身份书写与媒介动员——以《新青年》《中国青年》为中心的考察[J].传媒观察,2025(4):109-120.

[2]陈漱渝.《新青年》中的新文学史料[J].鲁迅研究月刊,2025(11):33-44.

[3]宋明炜.新青年的成长:“五四”与现代小说的诞生[J].四川大学学报（哲学社会科学版）,2025(1):120-131+234.

[4]秦素银.钱玄同与周氏兄弟关于《新青年》六卷二号往来通信考释[J].鲁迅研究月刊,2025(11):45-63.

[5]欧阳哲生.从《新青年》获取前进的智慧与勇气[J].鲁迅研究月刊,2025(9):5-6.
```
将txt格式内容，添加对excel的新增列`参考格式`中。顺序是对应的如 [1]就是excel中第一条数据（第二行），`[5]`就是第5条（第6行）数据的参考格式
