# 开发变更记录
- **日期**: 2026-05-05

## 1. 变更摘要
统一 `advanced-search` 的时间筛选参数为 `--date-from` 与 `--date-to`，移除旧的 `--year` 与 `--deadline-date`。
CLI 现在支持 `YYYY`、`YYYY-M(M)`、`YYYY-MM-DD` 三种输入粒度，并在内部归一化为 `YYYY-MM-DD`。
高级检索页面的发表时间填写逻辑改为统一写入 `#datebox0` 与 `#datebox1`，修复旧版“起始年/结束年”选择器失效的问题。
