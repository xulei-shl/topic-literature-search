# 开发变更记录
- **日期**: 2026-05-06

## 1. 变更摘要
修复维普高级检索在“目标数量小于每页条数”时可能误触发整页全选的问题。实现上保留原有部分页逐条勾选路径，只在结果页复选框收集阶段轻量过滤页级全选控件，并补充部分页勾选的显式日志；同时将部分页逐条勾选的执行节奏对齐到 `cnki-search`，改为页内连续勾选、页末一次计数校验，避免逐条读取 `topcount` 带来的性能问题和提前终止误判。

## 2. 文件清单
- `vp-search/scripts/interactor.py`: 修改结果页复选框过滤与部分页勾选日志逻辑
- `tests/test_vp_interactor.py`: 修正失效测试并新增全选控件过滤回归用例
- `docs/troubleshooting/20260506_维普部分页勾选误触发整页全选.md`: 新增故障复盘
