## 1. 变更摘要
- 修正 `vp-search` 批量勾选逻辑，当前页在满足整页目标时不再被限制为“仅批次第一页可全选”。
- 调整页级全选失败后的回退策略，改为撤销当前页全选并回退逐条勾选，避免直接清空全部已选结果。
- 新增逐条勾选的数量收敛逻辑，每勾选一条后读取页面“已选数量”，达到目标后立即停止。
- 补充 `vp-search` 交互测试，覆盖“第二页继续整页全选”“页级全选异常回退”“逐条勾选提前停止”场景。

## 2. 文件清单
- `vp-search/scripts/interactor.py`: 调整当前页整页全选判定与逐条勾选收敛逻辑。
- `tests/test_vp_interactor.py`: 新增并更新批量勾选相关测试用例。
