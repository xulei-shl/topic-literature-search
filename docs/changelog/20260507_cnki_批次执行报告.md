# 开发变更记录
- **日期**: 2026-05-07
- **对应设计文档**: 无（小功能扩展）

## 1. 变更摘要
- 为高级检索批量导出新增批次级文本报告输出。
- 每个批次导出完成后生成一个 `batch-report.txt`，记录检索词、状态、总数、选中数、计划导出数、批次进度、日期范围、核心筛选、页码范围、导出文件与进度文件。
- 整个任务完成后额外生成一个总报告 `report.txt`，便于直接查看本次执行结果。

## 2. 文件清单
- `src/core/advanced_export_flow.py`: 修改，新增批次报告与总报告生成逻辑，并将报告路径写回结果结构。
- `src/core/advanced_export_types.py`: 修改，补充报告文件与批次页码范围字段。
- `src/utils/result_output.py`: 修改，新增总报告文件名生成工具。
- `cnki-search/scripts/cnki_selection_ops.py`: 修改，记录单批次页码起止范围。
- `cnki-search/scripts/utils.py`: 修改，在 CLI 可读输出中展示总报告路径。
- `cnki-search/scripts/cli.py`: 修改，在 `saved_files` 中补充报告文件路径。
- `tests/test_advanced_export_flow.py`: 修改，验证批次报告和总报告文件生成。
- `tests/test_cnki_interactor.py`: 修改，验证批次页码范围写入勾选结果。
