## 1. 变更摘要
重构 `vp-search` 的高级检索勾选与导出主链路，使其整体行为尽量对齐 `cnki-search`、`wanfang-search` 的成功实现。

本次调整重点包括：
- 勾选主流程改为按“当前页实际勾选数量”推进
- 减少对全局已选数量 DOM 读取的依赖
- 第一页切片过滤后支持顺延补齐，避免回退到全量可见性扫描
- 去掉导出页打开时对 `already_at_target` 的缩短超时逻辑
- 将导出入口失败拆分为“未打开”和“未就绪”两类判定
- `Excel导出` / `参考文献` 统一改为“切换类型标签后点击 `#exportbtn` 下载”
- 若下载结果类型与目标不一致，强制重切目标标签并重试一次

## 2. 文件清单
- `vp-search/scripts/vp_selection_ops.py`: 重构批次勾选主流程与页内实际勾选计数
- `vp-search/scripts/vp_export_ops.py`: 重构导出页打开与就绪判定逻辑
- `tests/test_vp_interactor.py`: 调整勾选回归测试，适配页内实勾推进模型
- `docs/troubleshooting/20260507_vp_勾选与导出链路重构复盘.md`: 新增故障复盘
