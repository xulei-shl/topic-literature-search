# 开发变更记录
- **日期**: 2026-05-05
- **对应设计文档**: [检索超时环境变量配置](/E:/Desk/xinqingnian/docs/changelog/20260505_检索超时环境变量配置.md)

## 1. 变更摘要
- 将 CNKI 与维普脚本中重复的超时配置读取逻辑收敛为共享工具。
- 超时配置改为统一走“显式参数 > 进程环境变量 > `.env` 文件 > 默认值”优先级。
- 配置解析不再为了读取超时值而回写全局 `os.environ`，减少脚本间副作用。

## 2. 文件清单
- `src/utils/search_timeout.py`: 新增，统一 `.env` 查找、解析与超时值计算。
- `src/utils/__init__.py`: 新增工具包标记文件。
- `cnki-search/scripts/config.py`: 修改，改为复用共享超时配置工具。
- `vp-search/scripts/config.py`: 修改，改为复用共享超时配置工具。
- `tests/test_search_timeout_config.py`: 修改，补充“进程环境变量优先于 `.env`”测试。
