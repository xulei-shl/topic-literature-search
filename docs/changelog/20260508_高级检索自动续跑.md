# 开发变更记录
- **日期**: 2026-05-08

## 1. 变更摘要
为 `CNKI`、`VP`、`万方` 三个高级检索模块增加自动续跑能力：

- 未显式传入 `--progress-file` 时，若默认 `progress-*.json` 已存在，会自动加载历史进度继续执行。
- `advanced-search` 执行过程中遇到运行时异常时，会关闭当前浏览器并自动发起下一轮执行，继续基于进度文件续跑。
- 参数错误、进度文件冲突、用户中断等非可恢复场景仍保持原有退出行为，避免无效重试。
